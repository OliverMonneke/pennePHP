Welcome to pennePHP.

What is pennePHP? Well, pennePHP is a collection of helpfull algorithms to speed up coding. You can easily manage different datatypes, have a collection of exceptionsor filesystem operations. It uses composer as PHP packaging system. How to install: 

$ composer require codersquad/pennephp
    $ composer install

If you're need help using composer, please refer to the composer documentation.
